Answers for the questions

Describe your Javascript learning path (books you've read, courses you've taken, sample projects you've written etc.)

My first knowledge of Javascript started in May 2015. I’ve finished all free interactive courses on such online platform as Codeacadamy and Codeschool. These courses gave me a strong foundation in the JavaScript language so I was ready to move up to the next level.

I think the best resource for beginner to learn this language I’ve even seen is the Ilya Kantor’s website - learn.javascript.ru. I spend a lot of time on it. From July, 2015 to January, 2016 I was a student at GoIT. It was a fantastic Frond-End Development course, where I had ability to practice my knowledge and receive teacher’s help in this.

My diving deeper into Javascript began from workshop “Advanced Javascript by Kyle Simpson” which I watched on plurasight.com. There I gained an advanced understanding of the core mechanics of how JavaScript compiles, optimizes, and executes in the browser.

I follow such Javascript’s Guru as Douglas Crockford, Nicholas Zakas, Eric Elliot, Addy Osmany, Kyle Sympson, Ryan Dahl.

Now I’m reading two interesting books:

  • Learning JavaScript Design Patterns. Addy Osmany.
  • JavaScript Patterns. Stoyan Stefanov.

At that moment I am studying online courses:

  • Coursera. Server-side Development with NodeJS
  • Coursera. Front-End JavaScript Frameworks: AngularJS
  • MongoDB University. MongoDB for node.js developers

I don’t list plural articles, videos on youtube which are useful to read to know about the latest news and tends in Javascript community.

Describe the most sophisticated task in Javascript that you solved

I think the most sophisticated task in Javascript for me was the task to write a little game for exam project in GoIT. The task is not difficult, but it was the first complex challenge, where I could develop all the logic myself. This helped me to understand more deeply how Javascript allows us to have richer interaction with web page, how to work with object prototype model, clojures, event loop and other features and advantages of Javascript language.
